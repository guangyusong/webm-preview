# Changelog

## 0.1.2

- Prefer the workspace extension host so compatibility previews can run against Remote SSH/Codespaces files.
- Allow the desktop fallback to prepare previews for `vscode-remote` file resources.

## 0.1.1

- Corrected the license copyright attribution to Guangyu Song.
- Kept the published display name as WebM Video Viewer for Marketplace compatibility.

## 0.1.0

- Added direct `.webm` playback in a read-only custom editor.
- Added a desktop-only compatibility fallback that generates cached H.264 + MP3 previews after direct playback fails.
- Added cache size controls and a clear-cache command.
- Added generated smoke-test fixtures, packaging metadata, and Marketplace-ready assets.
